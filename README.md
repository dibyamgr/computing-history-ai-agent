# Computing History Agent

An AI-powered agent that explores the history of computing, from early mechanical calculators to modern artificial intelligence systems.

## Features

- Interactive AI agent for computing history
- Timeline-based exploration
- Historical figures and inventions
- Evolution of programming languages
- Computer hardware milestones
- AI and machine learning history
- Natural language question answering

## Example Questions

- Who invented the first programmable computer?
- How did UNIX influence modern operating systems?
- What led to the rise of personal computers?
- How did the Internet evolve?
- What is the history of artificial intelligence?

## Tech Stack

- TypeScript
- Node.js
- AI Agent Framework
- Azure AI Foundry
- OpenAI Models
